# Claude Code Test

A collection of browser-based games built with HTML5 Canvas and vanilla JavaScript. No dependencies, no build tools — just open the files in your browser.

## Games

### Retro Top-Down Shooter (`shooter.html`)

A top-down 2D shooter with retro pixel-art style graphics.

**Controls:**
- Arrow keys / WASD — Move
- Mouse — Aim
- Click / Hold — Shoot

**Features:**
- Multi-color pixel-art player with 4-frame walk animation
- Gun with muzzle flash, screen shake, and particle effects
- 3 enemy types: basic, fast, tank (+ bosses from level 5)
- Level progression with kill-count system
- Health pickups from defeated enemies
- Death animation with pixel scatter effect
- Menu, level-up, and game over screens

### Tic Tac Toe (`tictactoe.html`)

A simple two-player tic-tac-toe game with a dark theme.

**Features:**
- X (red) vs O (blue)
- Win detection with pulsing animation
- Restart button

## Running

Open any `.html` file directly in your browser. No server required.
